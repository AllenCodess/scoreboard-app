# Scoreboard App

This app allows a team to score up to a certain amount of points before resetting the game.

Touchdown +6
Field Goal + 3
Extra Point +1

Once a team reaches 42 buttons disable and you have to reset the game

## Tech Stack

HTML, CSS, JAVASCRIPT
